CRM Student Management app for University of Oxford

A mission-critical student management ecosystem for the University of Oxford

Project Overview

A high-concurrency, bespoke CRM and ERP platform architected with Python, Django, and AWS to centralize complex university operations. Key Features: • Student Lifecycle & Enrollment: End-to-end management of the student journey from initial registration to module completion and sending out course completion certificates. • Tutor & Faculty Management: Integrated scheduling and resource allocation of tutors to modules. • Service Orchestration: Centralized hub managing other 200+ distinct university services and academic modules. • High-Availability Architecture: Built to support thousands of concurrent users with rigorous data integrity and uptime standards. • Complex Data Modeling: Engineered custom logic to handle multi-faceted relationships between academic programs, staff, and students. • Architectural design: The apps connects to several other internal applications via api and other 3rd party applications. This project demonstrates the ability to architect and scale mission-critical enterprise software within a world-class institution. It features a robust backend capable of handling deep relational data and a secure infrastructure designed for seamless institutional interoperability.
